How to Use

HTML Structure

<div class="image-gallery">
    <div class="gallery-main-image">
        <img src="main-image.jpg" alt="Main Image" />
    </div>
    <div class="gallery-caption">Default caption</div>
    <div class="gallery-thumbnails">
        <div class="gallery-thumbnail">
            <img src="thumb1.jpg" alt="Image 1" data-caption="Caption 1" />
        </div>
        <div class="gallery-thumbnail">
            <img src="thumb2.jpg" alt="Image 2" data-caption="Caption 2" />
        </div>
    </div>
</div>

Key Features

  • Add data-caption attribute to thumbnail images for dynamic captions
  • Use data-fullsize attribute if you want different full-size images
  • The gallery automatically initializes when the page loads
  • Fully responsive and accessible out of the box
